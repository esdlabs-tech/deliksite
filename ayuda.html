¿Necesitas ayuda?

Bienvenido a la página de ayuda. Aquí encontrarás respuestas a las preguntas más frecuentes y guías para resolver problemas comunes.

Preguntas Frecuentes

  • ¿Cómo puedo descargar los programas?
  • Solamente necesitas tener instalado el Software Chocolatey, luego, selecciona los programas que quieres instalar y selecciona descargar, lo que activa un prompt con el comando que has de copiar en tu terminal.
  • No puedo copiar el código
    • Si no puedes copiar el código, puedes probar con otro navegador web, Microsoft Edge da algunos errores.
  • ¿Cómo puedo contactar?
  • Para contactar con nosotros, puedes hacerlo a través de la página de contacto, donde encontrarás un formulario para rellenar.
